На позапрошлой работе собирал приббор предотвращающий мухляж со стороны клиентов детского развлекательного центра. 
Клиенты засовывали руки в фотодатчики, набивая баллы. Решил эту проблемму следующим способом -  
собрав прибор анализирующий и в случае необходимости блокирующий датчики. 

Схема следит за датчиками, и если замечает повышенную активность блокирует их. 
Датчик при сработке замыкает сигналбный контакт на землю, выход схемы подключается
паралельно, и при блокировке, то-же замыкает его на землю, тем самым аппарат перестает
реагировать на сигналы датчика. Настроена система следующим образом. Постоянно считаем
импульсы, каждые две секунды смотрим сколько было импульсов с прошлой проверки, если на
каком-то из каналов импульсов больше чем 5, то блокируем этот канал на 20 секунд. Опытным
путем проверено что при честной игре за 2 секунды через голову больше 5 шаров не проходит,
только при набивании руками. Особенностью схемы является то что вход и выход совмещены и 
для интеграции схемы нужно подключить по одному проводу к каждому датчику и два провода на 
питание. У нас аппарат работает с отключенными нижними головами, поэтому схема 2-х канальная.
Для работы в 4 канальном режиме нужно будет полностью переписывать код, т.к. работа организована
с помощю аппаратных прерываний,а каналов прерываний у arduino nano только 2. Схема показала себя 
работоспособной, не глючит, повышенная выдача тикетов прекратилась..
